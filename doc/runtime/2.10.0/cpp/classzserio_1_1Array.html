Detailed Description

template<typename RAW_ARRAY, typename ARRAY_TRAITS, ArrayType ARRAY_TYPE, typename OFFSET_CHECKER = detail::DummyOffsetChecker, typename OFFSET_INITIALIZER = detail::DummyOffsetInitializer>
class zserio::Array< RAW_ARRAY, ARRAY_TRAITS, ARRAY_TYPE, OFFSET_CHECKER, OFFSET_INITIALIZER >

Array wrapper for zserio arrays.

The wrapper is used to encapsulate logic of operations with zserio arrays.
