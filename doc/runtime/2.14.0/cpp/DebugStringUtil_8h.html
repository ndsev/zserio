Detailed Description

It provides utilities for JSON debug string which can be obtained from zserio objects.

These utilities are not used by generated code and they are provided only for user convenience.

Note
Please note that Zserio objects must be generated with -withTypeInfoCode and -withReflectionCode zserio options to enable JSON debug string!
Please note that file operations allocate memory as needed and are not designed to use allocators.
